每个思想都有固定的解答模板。。比如回溯：

```javascript
int a[n];
    try(int i)
    {
        if(i>n)
           输出结果;
         else
        {
           for(j = 下界; j <= 上界; j=j+1)  // 枚举i所有可能的路径
           {
              if(fun(j))                 // 满足限界函数和约束条件
                {
                   a[i] = j;
                    ...                         // 其他操作
                   try(i+1);
                 回溯前的清理工作（如a[i]置空值等）;
                 }
            }
        }
   }
```
